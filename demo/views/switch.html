Switch

Análise

O componente interruptor (switch) é utilizado quando há necessidade de alternar um conteúdo através de um menu pré-estabelecido. Dependendo do comportamento ela possui um visual diferente. Por exemplo: Se adicionarmos o comportamento Tab ele atuará com Abas, se colocar List ele atuara como uma lista ou tera um efeito acordeão.

Navegação

A navegaçao é definida com um elemento mv-switch-nav ou classe respectiva. Dentro dele deve estar uma lista (geralmente ul) contendo os links

Botões

Uso

Pode ser utilizado tanto como atributo mv-btn como classe .mv-btn. Recomenda-se utilizar com o elemento button. Veja o exemplo abaixo:

<button mv-btn>Default</button>

Cor data-color

Para cada estado existe um botão respectivo. Basta adicionar a classe com nome na cor.

Ao utilizar o atributo color

Componente Cor Descrição
default Botão padrão
info Botão de informações
error Botão de operações realizadas com erros
success Botão de operações realizadas com sucesso
warning Botão de atenção
muted Botão desativado

Ícones

mv-icon [restrict: (A)]

Pode ser utilizado tanto como atributo mv-icon como classe .mv-icon. Recomenda-se utilizar com o elemento i.

Atributos

  • name *obrigatório

    Nome que será utilizado no ícone segundo uma tabela definida com os ícones cadastrados

  • fixed

    Define ícones com uma largura fixa. Ótimo para usar quando diferentes larguras de ícone estão fora de alinhamento. Especialmente útil em coisas como listas de navegação e grupos de listas.

  • flip [h,v,a] {default:h}

    Cria efeito espelhamento tanto na vertical e/ou na horizontal. Com a opção h para horizontal, v para vertical e a para ambos.

  • inverse

    Altera para uma cor invertida. Usualmente utilizado em pilha de ícones.

  • prefix {default: fa}

    Prefixo do recurso de fontes utilizada. Por padrão é utilizado fa por causa da inclusão do FontAwesome. Mas pode ser alterado conforme a biblioteca de ícones de sua escolha.

  • rotate [90, 180, 270]

    Rotaciona o ícone em 90º, 180º ou 270º

  • spin

    Efetua uma animação onde o ícone fica girando em seu eixo central.

  • zoom [lg, 2x, 3x, 4x, 5x]

    Aumenta o tamanho do ícone

Uso

<i mv-icon name="user"><i/>

Recomenda-se utilizar com o elemento ul.

<ul mv-icon-list>
  [...]
</ul>

mv-icon-stack [restrict: (A)]

Recomenda-se utilizar com o elemento span.

Uso

<span mv-icon-stack grow="false">
  <i mv-icon name="user"></i>
  <i mv-icon name="ban" class="text-error"></i>
</span>

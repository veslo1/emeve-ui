Layout

mv-container [restrict: (EA)]

Implementa um container básico para layout. Por padrão tem característica de exterder 100% da tela horizontalmente, ou melhor, é fluido.

Atributos

  • fluid

    Altera o modo de operação para fluid, assim o seu comprimento é ajustado ao conteúdo da tela do dispositivo.

  • mode [static,fluid] {default:fluid}

    Altera o modo de comprimento para fluid (porcentagem) ou static(definido para cada resolução).

  • static

    Altera o modo de operação para static

Implementação

Implementando por elemento:

<mv-container>
  [...]
</mv-container>

Implementando por classe:

<div class="mv-container">
  [...]
</div>

mv-row [restrict: (EA)]

Insere linhas no layout da grade. Recomenda-se utilizar o elemento div caso seja implementado a diretiva por classe.

Atributos

  • fill

    Define que todas as colunas tenham o mesmo tamanho.

Implementação

Implementado por elemento:

<mv-container>
  <mv-row>
    [...]
  </mv-row>
</mv-container>

Implementando por classe

<div class="mv-container">
  <div class="mv-row">
    [...]
  </div>
</div>

mv-col [restrict: (EA)]

Insere colunas no layout da grade. Recomenda-se utilizar o elemento div caso seja implementado a diretiva por classe.

Tamanhos

Cada tamanho se refere a uma largura de tela do dispositivo. Os tamanhos definidos são para o modo strict:

Tela Tamanho Prefixo Dispositivo
Extra Small até 768px xs Mobile
Small 768px sm Tablet
Medium 992px md Desktop
Large 1200px lg Wide desktop

Atributos

  • size

    Adiciona valores aos comprimentos de cada dispositivo. Ex.: {xs:1,sm:2,md:4,lg:8}.

  • pull

    Puxa a coluna em x. Ex.: {lg:8}. Assim a coluna será puxada 8 colunas em dispositivos wide desktop

  • push

    Empurra a coluna. Ex.: {sm:3}. Assim a coluna será empurrada 3 colunas em dispositivos mobile

Implementação

Implementado por elemento:

<mv-container>
  <mv-row>
    <mv-col size="{xs:6, sm:3,md:4,lg:2}">

    </mv-col>
  </mv-row>
</mv-container>

Implementando por classe

<div class="mv-container">
  <div class="mv-row">
    <div class="mv-col xs-6 sm-3 md-4 lg-2">
      [...]
    </div>
  </div>
</div>
